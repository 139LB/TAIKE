微信分享指南

重要提示

微信无法直接分享本地链接（localhost），需要将网站部署到公网才能分享。以下是几种分享方案：

1 最简单方案：部署到免费托管平台

  1. 访问 GitHub Pages 或 Vercel
  2. 注册账号并创建新项目
  3. 上传你的网站文件（所有HTML文件）
  4. 获得一个公网链接，例如：https://yourname.github.io/website/
  5. 将这个链接分享到微信

2 内网穿透方案（需要技术）

  1. 下载并安装内网穿透工具（如ngrok、frp）
  2. 运行命令将本地8000端口映射到公网
  3. 获得一个临时公网链接
  4. 将链接分享到微信
ngrok http 8000
注意：ngrok需要注册账号，免费版本链接会变化，每次重启都会生成新链接。

3 云服务器方案（长期稳定）

  1. 购买或租用云服务器（阿里云、腾讯云等）
  2. 配置Web服务器（Nginx、Apache等）
  3. 上传网站文件到服务器
  4. 获得固定域名和公网IP
  5. 将域名分享到微信

4 同一WiFi局域网分享

  1. 确保手机和电脑在同一WiFi网络
  2. 在电脑上打开命令提示符（CMD）
  3. 输入命令查看IP地址：
ipconfig
  1. 找到"IPv4 地址"，例如：192.168.1.100
  2. 在手机浏览器中输入：http://192.168.1.100:8000
  3. 将这个链接分享到微信（仅限同一WiFi）

推荐方案

对于快速分享，推荐使用GitHub Pages或Vercel，它们完全免费，操作简单，且生成的链接稳定。
